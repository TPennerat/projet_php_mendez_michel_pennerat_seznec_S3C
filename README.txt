Quelques modifications par rapport au sujet initial :
Pour éviter les redondances dans la BDD, nous avons choisi de dissocier l'objet item des listes. C'est à dire qu'un item peut-être dans 0 à n listes.
Pour impléemnter cette modification, nous avons du modifier certaines consignes du sujet; par exemple, toutes les informations telles que la réservation ou les cagnottes sont stockées dans la table item_liste.
